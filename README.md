# koa2-blog

#### 介绍
基于koa2的个人博客，已更新至2.0版本

#### 软件架构
1.  koa2
2.  sequelize
3.  mysql
4.  redis
5.  jwt
6.  lin-validator


#### 安装教程
1.  安装：
```js
npm install // 推荐使用 yarn安装
```
2.  运行：
```js
// 运行1.0版本请运行
npm run start:v1
// 运行2.0版本请运行
npm run start:v2
```
